Dicionário HTML

Tags são estruturas de linguagem de marcação contendo instruções. São utilizadas para informar ao navegador que tipo de estrutura é essa que está sendo construída, podendo ser títulos, parágrafos, imagens, links, entre outros.

Principais TAG's

Tag's h1 - h6

As tags h1 ao h6 são usadas para definir títulos HTML, sendo h1 o principal nível de título e h6 o nível mais baixo.

Tag p

O elemento "p" representa um parágrafo.

Tag br

A tag "br" representa quebra de linha.

Tag ul

A tag "ul" representa uma lista não ordenada.

Tag ol

A tag "ol" representa uma lista ordenada.

Tag li

A Tag "li" representa os elementos dessa lista.

Tag body

Body representa todo conteúdo do site.

Tag main

Main define o conteúdo principal.

Tag header

Header representa conteúdo introdutório. Ex.: cabeçalho, navegação do site.

Tag div

Uma "div" representa uma divisão dentro do documento HTML.

Tag span

Span é um contêiner embutido usado para marcar uma parte de um texto ou uma parte de um documento.

Tag form

"Form" é usada para criar um formulário HTML para entrada do usuário.

Tag fieldset

A tag "fieldset" é usada para agrupar elementos relacionados em um formulário, desenhando uma caixa ao redor.

Tag legend

Legend define uma legenda para o elemento "fieldset".

Tag label

Label define um rótulo para vários elementos.

Tag input

"Input" especifica um campo de entrada onde o usuário pode inserir dados.

Tag select

"Select" elemento é usado para criar uma lista suspensa.

Tag option

"Option" define uma opção em uma lista de seleção.

Tag textarea

"Textarea" tag que define um controle de entrada de texto multilinha.

Tag footer

Footer: define um rodapé para um documento ou seção.

Tag figure

A tag "figure" especifica conteúdo independente, como fotos e diagramas. Também podemos utilizar a tag "img" para incorporar uma imagem em uma página HTML
